FEATURES

六大核心功能

自然语言编程

用中文描述检测需求，AI 自动分析语义，生成完整的节点化工作流方案。支持追问修改，实现「对话式」开发体验。

机器视觉算法库

内置模板匹配、Blob 分析、边缘检测、颜色识别、深度学习推理等 200+ 算法节点，覆盖主流检测场景。

节点画布编辑器

直观的拖拽式节点画布，支持节点组合、子流程封装、条件分支和循环控制，搭建复杂工作流如同搭积木。

设备通信

支持 Modbus TCP/RTU、OPC-UA、S7、TCP/IP、串口等主流工业协议，一键对接 PLC、传感器和执行器。

数据分析仪表盘

实时监控检测数据，内置 SPC 统计分析、良率趋势图、异常报警。支持执行日志追溯和报表导出。

插件扩展系统

开放式插件架构，支持第三方节点开发。提供完善的 SDK 和 API 文档，可灵活对接 MES、ERP 等企业系统。
